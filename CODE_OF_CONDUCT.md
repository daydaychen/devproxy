# 行为准则

## 我们的承诺

为了营造一个开放包容的社区，我们承诺确保所有参与者都能免受骚扰，无论其年龄、体型、可见或不可见的残障、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向。

我们承诺以促进社区友好、包容的方式行事和互动。

## 我们的标准

有助于积极环境的行为包括：

- 使用友善和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用性暗示语言或图像
-  trolling、侮辱/贬损评论、人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他可以被合理地认为不专业的行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并对任何不可接受行为的情况采取适当和公正的纠正措施。

项目维护者有权和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题和其他贡献，或临时或永久禁止任何贡献者从事其认为不适当、威胁、冒犯或有害的行为。

## 适用范围

本行为准则适用于所有项目空间和公共场所，当个人代表项目或其社区时。代表项目或社区的示例包括使用官方项目电子邮件地址、通过官方社交媒体账号发布，或在线上或线下活动中指定代表。代表性的定义可由项目维护者进一步澄清和定义。

## 执行

如遇滥用、骚扰或其他不可接受的行为，请通过以下方式联系项目团队：

- 打开 issue 报告
- 发送邮件至项目维护者

所有投诉将得到及时和公平的审查和调查。

项目维护者有义务对事件报告者的隐私和安全性保密。

## 处罚措施

如果项目维护者认定某行为构成违反本行为准则，他们可采取以下措施：

- 书面警告
- 暂时禁止
- 永久禁止
